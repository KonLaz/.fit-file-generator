# Wahoo Workout Builder

Structured cycling workout builder with FIT export for Wahoo-compatible devices.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Project Docs

This repo keeps the root `README.md` minimal. Detailed docs live in `docs/`:

- `docs/PROJECT_APPROACH.md`
- `docs/PHASE2_THREAD_HANDOFF.md`
- `docs/security_best_practices_report.md`

Important: `/docs` is intentionally listed in `.gitignore`, so these files are local-only and not committed by default.

## Scripts

- `npm run dev` - start local dev server
- `npm run lint` - lint TypeScript/React
- `npm run build` - production build check
